ImageZoom Plugin — Manual Test

Click an image to toggle zoom. Move mouse / drag to pan. Move cursor off-image to reset.

Single image — default zoom (150%)

$('#img-default').imageZoom();

Single image — fullWidth

$('#img-fullwidth').imageZoom({ fullWidth: true });

Single image — custom zoom (200%)

$('#img-200').imageZoom({ zoom: 200 });

Single image — custom zoom (300%)

$('#img-300').imageZoom({ zoom: 300 });

Gallery — multiple images with same class

Each image has its own zoom state (no cross-bleeding).
